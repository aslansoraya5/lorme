# 📱 Download Lorme for Android

## 🚀 Three Ways to Get the Android App

---

## Option 1: Quick Build (Recommended for Developers)

### What You Need:
- Node.js installed
- ~10 minutes
- Internet connection

### Steps:

**Windows:**
```bash
# 1. Download this project
# 2. Open Command Prompt in project folder
# 3. Run:
build-android.bat
```

**Mac/Linux:**
```bash
# 1. Download this project
# 2. Open Terminal in project folder
# 3. Run:
chmod +x build-android.sh
./build-android.sh
```

**Result:** APK ready at `android/app/build/outputs/apk/debug/app-debug.apk`

---

## Option 2: PWA Builder (No Coding Required)

### Steps:

1. **Host the web app** (or use the dist folder):
   - Upload `dist/` folder to any web host
   - Or use GitHub Pages, Netlify, Vercel (free)

2. **Visit PWABuilder:**
   - Go to https://www.pwabuilder.com

3. **Enter your URL:**
   - Type your website URL
   - Click "Start"

4. **Download Android Package:**
   - Click "Package for Android"
   - Download the APK
   - Install on your device!

**Time:** 5-10 minutes
**Difficulty:** Easy ⭐

## 📥 Installing the APK

### Step-by-Step:

1. **Transfer APK to phone:**
   - Email to yourself
   - Google Drive/Dropbox
   - USB cable
   - Bluetooth

2. **Enable installation:**
   ```
   Settings → Security → Unknown Sources (ON)
   ```
   Or on newer Android:
   ```
   Settings → Apps → Special Access → Install Unknown Apps → [Your File Manager] → Allow
   ```

3. **Install:**
   - Open file manager
   - Find the APK file
   - Tap to install
   - Open Lorme!

## 🆘 Need Help?

### Common Issues:

**"App won't install"**
- Enable "Unknown Sources" in settings
- Check if APK is corrupted (re-download)
- Clear Play Protect cache

**"App crashes on open"**
- Android version too old (need 5.1+)
- Reinstall the app
- Clear app data

**"How do I build the APK?"**
- See `ANDROID-README.md` for detailed instructions
- Or use Option 2 (PWA Builder) - easiest!
